OBD Explorer Demos

Interactive HTML demos for exploring structure in the Ordered Binomial Distribution (OBD).

Repository: github.com/dperlman/OBDExplorer

Available Variants

  • Variant 1 - E vs P / E vs N explorer with range controls and tie-point overlays.
  • Variant 2 - Dual-panel view: binomial distribution and PCA projection.
  • Variant 3 - Quad layout with binomial view, PCA view, and E vs p panel.
  • Variant 4 - PCA-only explorer with projection controls.
  • Variant 5 - Tie-field values vs n at selected tie index/range.
  • Variant 6 - Tie-field values vs tie index for selected n/range.
